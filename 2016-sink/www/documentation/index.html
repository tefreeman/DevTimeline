How to install?

Copy the entire folder to the Document Root of your web server which is usually named www or public_html.

Config a Tile

                $tile = array(
                    'name'        => 'about',
                    'thumbnail'   => 'images/about.png',
                    'content'     => 'About Us',
                    'url'         => 'about.php',
                    'size'        => '4x2',
                    'theme'       => 'red',
                    'link'        => ''
                )
  • name
    The unique name of every tile, displayed as a hash tag in URL when a tile is clicked. For example: http://www.yoursite.com/#about
  • thumbnail
    Relative path to the tile icon (64px X 64px or 128px X 128px)
  • content
    This is the short description shown at the left bottom corner of a tile
  • url
    When a tile is clicked, the tile content is loaed via XHR from a page. So, this is the URL of the page. For example: about.php
  • size
    Tile size. Possible values: 1x12x24x2
  • theme
    Pre-defined theme. Possible values: blueorangeredgreendarkgreenpurpledardreddardblueyellowgrey
  • colour
    Tile colur (Hex code, e.g. #000000).
  • link
    An external URL. When this is set, clicking a tile takes you to the URL in a new tab

Config a Tile Container

                $tile_container = array(
                    'size'    => 'full',
                    'theme'   => '',
                    'tiles'   => array($tiles, $tiles, ...)
                );
  • size
    The container size. Possible values: full half
  • theme
    Overall container colour. When conatiner colour is set to blue all its tiles will be red no matter what tile theme colour is.
    Possible values: blueorangeredgreendarkgreenpurpledardreddardblueyellowgrey
  • tiles
    An array of all the tiles in the container.

A little bit more...

  • Page content
    For instance you have an "about us" tile, and the tile url is "about_us.php". In "about_us.php", make sure you do not include <HTML><HEAD> or <BODY> tags as the entire page content is loaded via AJAX and set as innerHTML of a DOM. What you put in the page should be what you usually write in <BODY> of a normal web page.
    Note: page content is cached in memory once loaded.
  • Loading extra JavaScript and CSS
    Open ui.class.php in includes folder, and find the code below: 
                            // system default CSS, JS
                        $this->addCSS('metro.css');
                        $this->addCSS('metro_mobile.css', 'screen and (max-height: 500px), screen and (orientation:portrait)');
                        $this->addJavaScript('jquery.js');
                        $this->addJavaScript('metro.js');

                        // your own CSS, JS
                        $this->addCSS('example1.css');       // CSS
                        $this->addCSS('example2.css');       // CSS
                        ...
                        $this->addJavaScript('example1.js'); // JavaScript
                        $this->addJavaScript('example2.js'); // JavaScript
